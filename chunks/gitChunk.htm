GIT & UNIX Command Cheatsheet

Getting to Sandbox

1. cd /c - brings you to c:

2. cd sandbox - takes you to sandbox directory

UNIX Commands

ls - list files

ls -A - list files including hidden files

cp - copy

rm <file> - remove directory and all files

rm -r - remove and all files

rm -rf - remove directory and all files with no warnings

cd - change directory

../ - takes you up a directory

touch - createfile if it doesn't exist, change last modified date

GIT Commands

git init - bootstrap a repository

git clone <repourl> <folder> - clone a remote repository to a local folder

git status -s - show which files need to be committed

git add <filename> - stage a file to be committed

git add . -A - stage a file that was removed

git reset <filename> - unstage a file

git commit -m "<message>" - commit staged changes

git push <remote> <branch> - push local changes to server

git remote add <remote> <url> - add a shortcut to a remote server

git branch - list of branches

git branch <branch> - create a branch called name

git branch -d <branch> - delete a local branch called name

git push <remote> :<branch> - delete branch on server

git checkout <branch> - check out branch <branch>

git diff <branch> - list differences between current branch and <branch>

git merge <branch> <branch> - merge this <branch> from that <branch>

notepad <filename> - open files in notepad
